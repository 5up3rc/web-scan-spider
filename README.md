使用说明：
=======================

代码是python脚本，spider.py是扫描脚本，主要找到某个网站的下属连接

wapiti2.py主要是对找到的url进行安全测试


1.修改spider.py
----------------------------
        URL = "http://www.example.com/"
        DOMAIN = "example.com"
改成你自己想扫描的网站和域名

2.扫描
----------------------------
        python spider.py


3.扫描结果
---------------------------
        扫描结果会生成result.txt在当前目录下


其他
--------------------------
        根据自己的需求改动脚本实现自己的需求
